2) Написать скрипт который выполнит автоматически пункты 3, 4, 5, 6, 7, 8, 13
Создаем файл с именем script1.sh с помощью команды vi
vi script1.sh
далее нажимаем insert и вносим след данные в файл

#!/bin/bash
cd f1
mkdir p1 p2 p3
cd p1
touch file1.txt file2.txt file3.txt file1.json file2.json
mkdir p4 p5 p6
ls
mv file1.json file1.txt ../p2

чтобы сохранить изменеия и файл нажимаем ESC :wq enter
запускаем скрипт с помощью команды sh
sh script1.sh
Скрипт необходимо создавать находясь в корневой папке
